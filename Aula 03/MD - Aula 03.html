Visualização de Dados Não-Convencioanis¶

  • Nos dados convencionais, cada instncia do problema representada por um nmero de atributos

  • Em dados no convencionais no existe essa regra clara, e muitas vezes necessrio transformar os dados em algum formato especfico para que seja possvel aplicar os mtodos existentes para dados convencionais, de forma a no precisar criar novos mtodos

  • A visualizao de dados no convencionais diversa e depende dos dados que estamos analisando
    • Em dados temporais comum utilizar o tempo no eixo das abcissas (ou eixo x)
    • Em dados sequenciais comum utilizar tambm o eixo das abcissas para definir a sequncia dos dados
    • Em dados geogrficos temporais comum os eixos das abcissas (eixo x) e das ordenadas (eixo y) para longitude e latitude, respectivamente, sendo que o tempo dificl de ser representado

Dados Temporais¶

  • Um conjunto de dados temporal univariado (ou uma srie temporal univariada) pode ser definido como sequncia temporalmente ordenada de valores de uma varivel

$$\mathbf{x} = \langle(t_1, x_1), (t_2, x_2), \ldots, (t_m, x_m) )\rangle$$

  • Um conjunto de dados temporal multivariado (ou uma srie temporal multivariada) pode ser definida como uma sequncia temporalmente ordenada de valores de mltiplas variveis

$$ \mathbf{x} = \begin{matrix} \langle(t_1, & x_{1,1}, & x_{1,2}, & \ldots, & x_{1,d}), \\ (t_2, & x_{2,1}, & x_{2,2}, & \ldots, & x_{2,d}), \\ \ldots \\ (t_m, & x_{m,1}, & x_{m,2}, & \ldots, & x_{m,d})\rangle \end{matrix} $$

Procurando minimos locais e máximos locais na série temporal¶

  • Para visualizar mnimos e mximos locais podemos usar as funes stat_valleys e stat_peaks. Essas funes tambm tem um parmetro chamado de span que por default 5.

  • Para stat_valleys isso significa que dado um ponto central, se os dois elementos esquerda e direita forem maiores do que o elemento central, ento ele um vale, isto considera uma janela de tamanho 5.

  • Para stat_peaks isso significa que dado um ponto central, se os dois elementos esquerda e direita forem menores do que o elemento central, ento ele um pico.

  • Ento quanto maior seja o valor de span maior a janela considerada e menos pontos sero picos ou vales.

In [10]:
library(ggpmisc)
ggplot(data=economics[1:100,], aes(x=date, y=uempmed)) + geom_line() + geom_point(size = 0.7) + stat_peaks(colour = "red") + stat_valleys(colour = "blue")
