Visualizacão de Várias instâncias (exemplos) de uma Variável Contínua ao Longo do Tempo¶

  • Quando temos vrias instncias de uma varivel continua, como no caso de vrios exames de eletrocardiograma de pacientes (considerando apenas uma srie temporal de cada um) tambm temos que colocar os dados em formato adequado para visualizao.

  • Por exemplo baixar o seguinte conjunto de dados: ECG200

  • ECG200 um conjunto de dados bem conhecido sobre dados de eletrocardiograma com duas classes: 1 (normal) e -1 (anormal)

  • Colocar os dados na pasta /data/ECG200/

Um conjunto de dados temporais multivariados com mltiplas instncias definido como um conjunto:

$$X = \{ \mathbf{x}_1, \mathbf{x}_2, \ldots, \mathbf{x}_n \}$$

$$ \mathbf{x} = \begin{matrix} \langle(t_1, & x_{1,1}, & x_{1,2}, & \ldots, & x_{1,d}), \\ (t_2, & x_{2,1}, & x_{2,2}, & \ldots, & x_{2,d}), \\ \ldots \\ (t_m, & x_{m,1}, & x_{m,2}, & \ldots, & x_{m,d})\rangle \end{matrix} $$

Em conjunto de dados supervisionados ainda temos o valor da classe ($y$). Neste caso, definimos o conjunto de dados da seguinte forma:

$$X = \{ (\mathbf{x}_1,y_1), (\mathbf{x}_2,y_2), \ldots, (\mathbf{x}_n,y_n) \}$$

Dados de Textos¶

  • Existem muitos conjunto de dados com texto, que podem ser extrados por exemplo do Wikipedia ou do Twitter para estudo. Em R tem bibliotecas disponves para extrair dados desse nesse formato. Em geral, iro precisar criar uma chave de acesso para poder usar esse servio.

  • Vamos estudar um conjunto de dados de anlise de sentimento em reviews de filmes. Um dataset foi disponibilizado pela plataforma Kaggle no endereo https://www.kaggle.com/c/sentiment-analysis-on-movie-reviews/data

  • Baixar os arquivos em uma pasta dentro do projeto e carregar o arquivo train.csv

  • O arquivo contm frases entradas de uma sentena geral por um extrator de frases a partir de sentenas.

  • Essas frases foram rotuladas com o Sentimento;

    • 0 negative
    • 1 somewhat negative
    • 2 neutral
    • 3 somewhat positive
    • 4 positive

Gráficos de Frequência de Palavras¶

  • Uma atividade comum em visualizao de dados textais extrair a frequncia de ocorrncia das palavras nos textos. Para isso usamos a funo strsplit para criar uma nova coluna com a palavra e depois criar uma coluna de frequncia
In [19]:
  1. 'A'
  2. 'series'
  3. 'of'
  4. 'escapades'
  5. 'demonstrating'
  6. 'the'
In [27]:
dt1 <- text[,.(word=unlist(strsplit(Phrase," ")) )]
dt2 <- dt1[, .(count = .N),by=.(word)][order(-count)]

Ao visualizar a frequncia de palavras em textos muito comum

In [26]:
ggplot(dt2, aes(x=count)) + geom_histogram()

`stat_bin()` using `bins = 30`. Pick better value with `binwidth`.
In [27]:
ggplot(dt2, aes(x=count)) + geom_histogram() + scale_x_log10()

`stat_bin()` using `bins = 30`. Pick better value with `binwidth`.
  • Visualizao de frequncias individuais das palavras, limitada a um nmero especfico de palavras mais frequntes
In [42]:
ggplot(dt2[1:20], aes(y=count, x=word )) + geom_bar(stat = "identity")
  • Tambm podemos ordenar as barras pelo valor da frequncia
In [45]:
ggplot(dt2[1:20], aes(y=count, x=reorder(word,-count) )) + geom_bar(stat = "identity")

Gráfico de Nuvem de Palavras (wordcloud)¶

  • Atualmente comum encontrar na Internet as chamadas wordclouds, que so formas de representar a frequncia de palavras, onde quanto maior a frequncia de uma palavra num conjunto de dados, maior o seu tamanho no grfico.

  • No exemplo das revises de filmes, vamos ver quais so as palavras mais frequntes em textos onde o sentimento 4 (positive) e onde o setimento 0 (negative)

  • Visualizao das 30 palavras mais frequentes em revises positivas (Sentiment 4)
In [197]:
options(repr.plot.width=10, repr.plot.height=7)
In [225]:
dt3 <- dt2[Sentiment == "4"][order(-count)][1:100]
wordcloud(dt3$word, sqrt(dt3$count))
